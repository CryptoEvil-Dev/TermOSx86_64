# reboot (Rust Extended Boot) Расширенный загрузчик на Rust

**reboot** - Расширенный 32-битный загрузчик на Rust.
- Поддерживает видеорежим 0x411E 1280x1024 пикселей.
- Шрифт IBM VGA.

# Концепция
**reboot** - Как ОС для кастомных программ-загрузчиков ОС и как гипервизор. Пользователь может загрузить свой кастомный загрузчик на флешку и параметризованно загружать любую ОС, вплоть до эмуляции ARM64 на x86.

# Сборка Rust реализации

**Сборка**:
```sh
make
```

**Сборка / Запуск**:
```sh
make run
```

**Очистка**:
```sh
cargo clean
```
