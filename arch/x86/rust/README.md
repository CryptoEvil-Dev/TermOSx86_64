# Rust реализация

Подход к разработке был полностью изменён. В отличии от реализации на C++, переход от Real Mode к Long Mode протекает плавно.
Изменён подход к загрузчику:
- Появилась 32-битная расширенная версия загрузчика, являющаяся мини-ОС на Rust.
- 64-биное ядро грузит 32-битный загрузчик, в который можно передать исполняемые файлы, используя которые можно загрузить основную ОС как угодно.

# Сборка Rust реализации

**Сборка**:
```sh
make
```

**Сборка / Запуск**:
```sh
make run
```

**Очистка**:
```sh
cargo clean
```
